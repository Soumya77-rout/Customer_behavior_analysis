📊 Customer Shopping Behavior Analysis
🔍 Overview

This project analyzes customer shopping behavior using transactional data to uncover patterns in spending, customer segments, and product preferences. The goal is to generate actionable insights that support data-driven business decisions.

📁 Dataset
Rows: 3,900
Columns: 18
Includes:
Customer demographics (age, gender, location, subscription status)
Purchase details (items, category, amount, season, etc.)
Behavioral data (discount usage, frequency, ratings, shipping type)
Missing values handled in review ratings column
🛠️ Tools & Technologies
Python: Data cleaning, EDA (Pandas, NumPy)
SQL (PostgreSQL/MySQL/SQL Server): Data analysis & querying
Power BI: Dashboard & visualization
Gamma: Presentation (PPT)
Excel/CSV: Data source
⚙️ Project Steps
Data Loading & Exploration
Imported dataset and analyzed structure using Python
Data Cleaning & Preparation
Handled missing values
Standardized columns
Created new features (age groups, frequency metrics)
SQL Analysis
Answered business questions using queries:
Revenue by gender
High-spending customers
Product performance
Customer segmentation
Dashboard Creation
Built an interactive Power BI dashboard for visualization
Reporting & Presentation
Created a detailed report
Built a presentation using Gamma
📊 Dashboard

The Power BI dashboard provides:

Customer distribution and KPIs (total customers, avg purchase, ratings)
Revenue trends by category and age group
Subscription vs non-subscription insights
Sales performance across product categories

📈 Results & Insights
Identified loyal, returning, and new customer segments
Found high-revenue age groups and product categories
Analyzed impact of discounts and subscriptions on revenue
Highlighted top-performing and discount-dependent products
